# Social Connect App

A complete social media mobile application built with React Native and Firebase.

## Features

- ✅ User Authentication (Login/Register)
- ✅ Create, Edit, Delete Posts
- ✅ Upload Images in Posts
- ✅ Follow/Unfollow Users
- ✅ Real-time Messaging
- ✅ User Profile with Picture, Name, Bio
- ✅ Search Users
- ✅ Bottom Navigation
- ✅ Real-time Updates

## Tech Stack

- React Native (Expo)
- TypeScript
- Firebase (Auth, Firestore, Storage)
- React Navigation
